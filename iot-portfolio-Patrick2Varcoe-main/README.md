# UFCFVK-15-2 Internet of Things Repository

This repository contains the starting code and documentation for the IoT Portfolio project. You should complete your solutions in this repository, commit and push by the deadline set. 

Three preparatory worksheets (prep-worksheets) are provided to help you get started with the project.

## Preparatory Worksheets

- [**prep-ws0:**](prep/prep-ws0/prep-worksheet0.md) Setting Up Your IoT Development Environment
- [**prep-ws1:**](prep/prep-ws1/prep-worksheet1.md ) Basic IoT Device Programming. Complete this before attempting worksheet 1.
- [**prep-ws2:**](prep/prep-ws2/prep-worksheet2.md) IoT Client-Server Communication. Complete this before attempting worksheet 2.

---

Complete worksheets 1 and 2 in the corresponding folders [*ws1*](/ws1/) and [*ws2*](/ws2/) as described in the Assessment Brief available in Blackboard.
